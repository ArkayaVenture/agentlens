AgentLens installs an always-on local command center for your Claude Code, Cowork & Cursor sessions — tokens, sub-agents, 4-way data provenance and an ULTRACODE strategy view — plus a self-enriching knowledge base. Fully local. No hooks. No telemetry.

⚡ Install the plugin See what it does
# in Claude Code or Cowork
/plugin marketplace add arkayaventure/agentlens
/plugin install agentlens@arkaya-plugins
/agentlens:setup-agentlens # stands up the dashboard on :7717

Everything you ship with AI, on one screen

AgentLens reads the session transcripts you already have and turns them into a live, drill-down command center — then keeps a cited knowledge base current for you.

📊

Live monitoring

Sessions, sub-agent fan-out, tool calls with pass/fail, durations and models — drill Overview → session → sub-agent → tool call.

🪙

Token economics

Tokens in/out, cache-hit %, top token-consuming sessions, token-by-tool attribution and cost-cutting tips.

🌐

4-way provenance

Every tool call tagged internal / firm MCP / external web / external LLM — know exactly where your data flows.

🕸️

Workflow brain

A force-directed graph of your workflows plus a capability topology of skills, agents and connectors.

⚡

ULTRACODE view

Detects fan-out, tournament, loop-until-done, adversarial and classify-and-act patterns in your real sessions.

📡

Self-enriching feed

Scheduled research loops keep a dated, source-cited digest of the AI ecosystem flowing into your brain.

Secure by construction

Plugins run with your privileges, so AgentLens minimises its surface — and shows you everything it does.

No hooks. Nothing runs automatically on Claude Code events.
No bundled MCP servers. The plugin adds no network connectors.
No telemetry. Nothing is sent anywhere by this tool.
Zero dependencies. Python standard library + one HTML file.
Loopback-only. The server binds 127.0.0.1 — never your network.
Auditable. Plain, unobfuscated code with a path-traversal-guarded API.
