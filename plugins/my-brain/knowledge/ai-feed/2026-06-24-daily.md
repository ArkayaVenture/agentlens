## Anthropic

**Claude Tag launches in Slack** — 2026-06-23 · feature · `high`
Anthropic introduced Claude Tag: Claude joins Slack channels as a taggable (@Claude) multiplayer teammate that builds memory over time, takes proactive ambient initiative, and works asynchronously. In beta for Claude Enterprise and Team, runs on Opus 4.8, and replaces the existing Claude in Slack app.
Source: https://www.anthropic.com/news/introducing-claude-tag

**Claude Code 2.1.186** — 2026-06-22 · tooling · `medium`
Adds `claude mcp login`/`logout` CLI commands to authenticate MCP servers without the interactive menu (with `--no-browser` SSH support), status filtering in the /workflows agent detail view, and a Skills section in the /plugin Installed tab. `!` bash commands now auto-trigger Claude to respond to their output; plus background-agent and subagent fixes.
Source: https://code.claude.com/docs/en/changelog

## OpenAI

**GPT-5.5-Cyber released to defenders (Daybreak)** — 2026-06-22 · model · `high`
OpenAI expanded its Daybreak program with an updated Codex Security plugin and the full release of GPT-5.5-Cyber to trusted defenders, setting a new SOTA 85.6% on CyberGym (vs 81.8% for GPT-5.5), aimed at moving past vulnerability discovery toward end-to-end patch automation.
Source: https://openai.com/index/daybreak-securing-the-world/

**"Patch the Planet" initiative** — 2026-06-22 · safety · `medium`
A Daybreak initiative with Trail of Bits and HackerOne funding researchers to use Codex Security to help open-source maintainers turn findings into merged fixes. More than 30 projects committed, including cURL, Go, Python, Sigstore and pyca/cryptography.
Source: https://openai.com/index/patch-the-planet/

## Mistral

**Mistral OCR 4** — 2026-06-23 · model · `high`
Released OCR 4 (`mistral-ocr-4-0`; `mistral-ocr-latest` now points to it), a SOTA document-understanding model adding paragraph-level bounding boxes, typed-block classification and inline confidence scores, with 170-language support and single-container self-hosting. Available via API at $4/1000 pages ($2 with Batch) through Mistral Studio, Amazon SageMaker and Microsoft Foundry.
Source: https://mistral.ai/news/ocr-4/

## NVIDIA

**Halos for Robotics** — 2026-06-22 · safety · `high`

The open-source Halos Outside-In Safety Blueprint is in early access on GitHub; Agility is the first robotics partner.
Source: https://nvidianews.nvidia.com/news/nvidia-announces-halos-for-robotics-the-industrys-first-full-stack-safety-system-for-physical-ai

Source: https://nvidianews.nvidia.com/news/nvidia-vera-rubin-delivers-world-class-supercomputers-for-science

**CUDA-X AI-for-science software (DAQIRI, ALCHEMI)** — 2026-06-22 · tooling · `medium`
At ISC High Performance 2026, NVIDIA introduced new CUDA-X AI-for-science software: the DAQIRI high-performance data-acquisition networking library (now on GitHub) and new ALCHEMI NIM microservices/toolkit for chemistry and materials discovery, with cuPhoton reference code coming this summer.
Source: https://blogs.nvidia.com/blog/ai-for-science-software-cuda/
